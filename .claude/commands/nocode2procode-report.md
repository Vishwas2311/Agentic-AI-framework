# /nocode2procode-report

Create the final NoCode2ProCode by TrustEngines report for a generated app.

## Required Behavior

1. Read generated reports from the current migration workspace.
2. Summarize generated files, confidence, QA, design quality, security, compliance, unsupported items, and remaining risks.
3. Produce or update `final_migration_confidence_report.md`.
4. Keep the summary short enough for human review and link to detailed artifacts.
