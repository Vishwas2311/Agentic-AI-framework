# /nocode2procode-replay

Build or update the NoCode2ProCode by TrustEngines replay dashboard.

## Required Behavior

Compare original source evidence with generated app evidence:

- screens
- visual diffs
- API parity
- data parity
- permission parity
- unsupported items
- source truth decisions
- human decisions
- test results

Produce `genesis_replay_dashboard.html`.

## Compatibility

`/genesis-replay` remains supported as an older alias.
