# /nocode2procode-image

Rebuild or improve UI from an uploaded screenshot, sketch, Figma export image, brand image, or mobile screenshot.

## Required Behavior

1. Treat uploaded images as source evidence, even when Magic mode is `from_scratch`.
2. Run design strategy first, then UI/UX Pro Max, 21st.dev Magic, Motion after rebuild, and design quality gate.
3. Preserve requested fidelity while improving UX, accessibility, responsiveness, and polish.
4. Produce `design_decision_report.json`, `visual_lock_spec.json`, generated UI files, and design QA reports.
