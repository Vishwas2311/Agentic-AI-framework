# Screenshot To UI Prompt

Use this prompt for screenshot, wireframe, or image input.

## Goal

Convert visual evidence into UI-IR, not directly into final React code.

## Required Output

Produce:

- layout tree
- text inventory
- component candidates
- design tokens
- responsive assumptions
- accessibility concerns
- confidence per region

## Rules

- Do not invent hidden business logic from a screenshot.
- Separate visible UI from inferred behavior.
- Mark unclear controls for human review.
