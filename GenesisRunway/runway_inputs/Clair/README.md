# CLAIR Modern Web

Vercel-ready Next.js App Router implementation generated by Genesis.

- React, TypeScript, Tailwind CSS
- Vercel Route Handlers under `app/api`
- Typed in-project demo models in `lib/types.ts`
- Mock/demo data in `lib/mock-data.ts`
- OpenAI-powered triage and document summaries when `OPENAI_API_KEY` is configured
- No external database, no Prisma, no PostgreSQL

Local run:

```powershell
npm install
npm run dev
```

Environment:

```env
OPENAI_API_KEY=
OPENAI_MODEL=gpt-4.1-mini
```

Use `.env.local` for local development. Keep the key server-only; never rename it to `NEXT_PUBLIC_OPENAI_API_KEY`.

Vercel deployment:

- Add `OPENAI_API_KEY` in Project Settings > Environment Variables.
- Add `OPENAI_MODEL` only if you want to override the default model.
- Redeploy after adding or changing environment variables.
