# raw_data

Put any non-visual migration source evidence here.

Examples:

- PowerApps packages or `.pa.yaml`
- BRD, PRD, PDF, DOCX, PPTX
- XML, JSON, YAML exports
- SQL, CSV, Excel, schemas
- API/OpenAPI/Postman files
- website notes or URLs

Genesis treats this folder as source-truth and business-logic evidence.
