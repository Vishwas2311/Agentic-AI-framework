# Frontend React Generator Prompt

Generate React frontend from `canonical_app_spec.json`, `DESIGN.md`, `design_tokens.json`, and `component_registry.json`.

## Required

- routes
- pages
- forms
- tables
- dashboards
- API hooks/clients
- loading/error/empty states
- Playwright smoke tests

## Rules

- Use shadcn/Tailwind component patterns where configured.
- Follow UI/UX Pro Max design direction.
- Use Framer Motion only for purposeful interaction feedback.
- Ensure mobile and desktop layouts do not overlap or clip text.
