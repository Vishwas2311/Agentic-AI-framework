# /nocode2procode-website

Rebuild, modernize, or take inspiration from a website URL.

## Required Behavior

1. Use Firecrawl and Playwright before deciding final website/component structure.
2. Select Magic `website` for full pages/sites and Magic `component` for single cards, forms, navbars, or sections from the URL.
3. Use `get_inspired` for URL evidence and obey license, privacy, and protected media gates.
4. Improve the website beyond the source while preserving content/structure when the user asks for a rebuild.
