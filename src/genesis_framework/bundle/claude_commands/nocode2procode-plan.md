# /nocode2procode-plan

Plan the target architecture, source truth, data/API/UI mapping, test strategy, and deployment strategy.

## Required Behavior

1. Read discovery evidence and `.genesis` routing, policy, and capability files.
2. Decide architecture and migration strategy before generation.
3. Produce or update `canonical_app_spec.json`, `migration_contract.json`, and planning reports.
4. Treat extracted evidence as an input to planning, not as the final build decision by itself.
